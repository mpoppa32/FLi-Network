Intel Station — How to Use
Shared Intelligence
Both Mike and Bryce use the same app. Meetings logged by either person appear instantly for both. The green dot in the top bar means you are live and connected to the shared database.
Log Tab
Paste Otter.ai transcript with speaker labels for best results. Use Voice for post-meeting dictation (Chrome only). Hit Process and the AI structures everything in about 15 seconds.
Prep Tab
Enter who is in the room. AI pulls everything recorded about each person across all shared meetings and generates a one-page brief.
Setup for Bryce: Send him the IntelStation.html file. He opens it, enters his name (Bryce) and his own Anthropic API key, and he is immediately connected to the same shared database with all existing meetings already loaded.
